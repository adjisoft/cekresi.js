# Tutorial

## Pastiin
- Nodejs & npm diinstall
- Minimal node 18 keatas

- Clone
```sh
git clone https://github.com/adjisoft/cekresi.js
```

- Masuk
```sh
cd cekresi.js
```

## Jalankan

- Cek biasa
```sh
node cekresi.js SPXID0XXXXXXXXXXX --exp SPX #CEK BIASA
```

- Dump json
```sh
node cekresi.js SPXID0XXXXXXXXXXX --json 
```
